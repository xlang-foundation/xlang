# XLang™ Examples

The XLang codebase comes with a comprehensive suite of examples and test cases that demonstrate language features and API usage.

## Locating Examples

Under the `test/` folder and `test2026/` folder, you'll find numerous XLang and Python code examples. These act as living documentation for language semantics.

Our active test suites demonstrate:
- **Syntax & Language Features**: Control flows, tuple operations, native collections, dynamic typing operations.
- **Interoperability**: Examples showing how XLang integrates naturally with Python syntax and modules.
- **Modules**: Examples exploring built-in extensions such as SQLite interactions, YAML parsing, JSON loading, HTTP integrations, and threading.

*Note: While some older `.x` files may currently break as we continuously evolve the language, we are actively improving syntax alignment and expanding the standard library backward compatibility.*
